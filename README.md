# 🌿 Eco-Avatar: Guardián del Planeta Pro

¡Bienvenido al proyecto **Eco-Avatar**! Esta es una aplicación web interactiva diseñada para concientizar sobre el cuidado del medio ambiente a través de un avatar virtual reactivo (un simpático guardián de la naturaleza) que cambia de estado según las condiciones del ecosistema.

---

## 🚀 Características Principales

* **Biomas Dinámicos:** Cambia el fondo del entorno entre Bosque Vivo, Desierto Sol, Glaciar Alfa y Eco Cyber.
* **Sensores Reactivos:** Simulación de temperatura y humedad mediante barras deslizantes (*sliders*).
* **Estados de Ánimo Emocionales:** El avatar reacciona en tiempo real con videos dinámicos según el clima (Óptimo, Calor Crítico, Frío Extremo, Suelo Seco, Exceso de Humedad y Estado de Peligro).
* **Accesorios Interactivos:** Sistema de personalización para añadir ítems en pantalla (maceteros, escobas, gatos y perros) al lado del avatar.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica de la aplicación.
* **CSS3:** Diseño responsivo (adaptable a móviles y computadoras) y animaciones de efectos visuales.
* **JavaScript (Vanilla):** Lógica del sistema de sensores, control del reproductor de video reactivo y gestión de accesorios dinámicos.
* **Tabler Icons:** Pack de iconos modernos para la interfaz.

---

## 💻 Cómo Ejecutar el Proyecto Localmente

1. Descarga o clona este repositorio en tu computadora.
2. Asegúrate de tener los archivos multimedia de video (`.webm` o `.mp4`) en la misma carpeta raíz o configura los enlaces en línea dentro del archivo de scripts.
3. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Edge, Firefox, etc.).
4. ¡Empieza a interactuar con los sensores y personaliza tu guardián!
